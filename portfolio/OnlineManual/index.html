CML: Manually Checking Out Customers

How to check out customers in the event of system outages


This page will demonstrate a process CML circulation staff can use when the Polaris Integrated Library System (ILS) is down.

The process utilizes Microsoft Notepad for quick recording of barcodes in order to produce receipts that can be easily entered into the system when it comes back up.

There are links on the left to go directly to each step of the process. For more in depth information on Polaris and Notepad you can visit the links provided.

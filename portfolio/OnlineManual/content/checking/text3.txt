<span>Step 4: Print Receipt</span>
<ul>
<li>Click on file->print to print a receipt with the document contents.</li> <li>The printer is set by default to the receipt printer. After the receipt comes out, save this in a safe place for entering into the system at a later time.</li> <li>If customer desires a receipt, the same process can be repeated to provide them with a receipt.</li> <li>After completing the transaction highlight and clear all of the barcodes so the document is ready for the next customer</li></ul>
